## User Stories

### GenericUser
|Identifier|Name|Priority|Description|
| -------- | -- |:------:| --------- |
|US01      |About Page|medium|As a GenericUser, I want to access about page, so that I can see a complete website's description.|
|US02      |FAQ Page|medium|As a GenericUser, I want to access FAQ page, so that I can see a Frequently Asked Questions.|
|US03      |Contact Page|medium|As a GenericUser, I want to access contact page, so that I can see the website owners' contacts.|
|US04      |Terms Page|medium|As a GenericUser, I want to access terms and conditions page, so that I can see a complete list of terms and conditions imposed by the website.|


### UnauthenticatedUser
|Identifier|Name|Priority|Description|
| -------- | -- |:------:| --------- |
|US05      |Register|high|As an UnauthenticatedUser, I want to register myself into the system, so that I can login.|
|US06      |Register using external API|medium|As an UnauthenticatedUser, I want to register myself into the system using the facebook API, so that I can login.|
|US07      |Register using external API|medium|As an UnauthenticatedUser, I want to register myself into the system using the google API, so that I can login.|
|US08      |Log in|high|As an UnauthenticatedUser, I want to login into the system, so that I can access privileged information.|
|US09      |Log in using external API|medium|As an UnauthenticatedUser, I want to login into the system using the facebook API, so that I can access privileged information.|
|US10      |Log in using external API|medium|As an UnauthenticatedUser, I want to login into the system using the google API, so that I can access privileged information.|



### AuthenticatedUser
|Identifier|Name|Priority|Description|
| -------- | -- |:------:| --------- |
|US11      |Profile Page|high|As an AuthenticatedUser, I want to access my Profile Page and change my information, so that it stays up to date.|
|US12      |Feed Page|high|As an AuthenticatedUser, I want to access the Feed Page, so that I can see posts from the users/bands I follow.|
|US13      |Follow Users|high|As an AuthenticatedUser, I want to follow other users, so that I can see their posts on my feed page.|
|US14      |Unfollow Users|high|As an AuthenticatedUser, I want to unfollow other users, so that I no longer see their posts on my feed page.|
|US15      |Follow Bands|high|As an AuthenticatedUser, I want to follow bands, so that I can see their posts on my feed page.|
|US16      |Unfollow Bands|high|As an AuthenticatedUser, I want to unfollow bands, so that I no longer see their posts on my feed page.|
|US17      |Create Band Page|high|As an AuthenticatedUser, I want to create bands' pages, so that I can share my band with other people.|
|US18      |Apply to Bands|low|As an AuthenticatedUser, I want to apply to bands, so I can join a band looking for a musician.|
|US19      |Post|high|As an AuthenticatedUser, I want to post on the system, so that I can share content.|
|US20      |Post using external API|low|As an AuthenticatedUser, I want to post songs on the system using the soundcloud API, so that I can share content from soundcloud.|
|US21      |Post using external API|low|As an AuthenticatedUser, I want to post songs on the system using the youtube API, so that I can share content from youtube.|
|US22      |Comment|medium|As an AuthenticatedUser, I want to comment on posts from users I follow, so that I can share my opinion.|
|US23      |Comment|medium|As an AuthenticatedUser, I want to comment on posts from bands I follow, so that I can share my opinion.|
|US24      |Remove own posts|low|As an AuthenticatedUser, I want to remove my posts from the system, so that I can remove unnecessary posts.|
|US25      |Remove own comments|low|As an AuthenticatedUser, I want to remove my comments from the system, so that I can remove unnecessary comments.|
|US26      |Report bands'|medium|As an AuthenticatedUser, I want to report bands' behaviour, so that negative bands or negative content gets removed from the system.|
|US27      |Report users'|medium|As an AuthenticatedUser, I want to report other users' behaviour, so that negative users or negative content gets removed from the system.|
|US28      |Rate bands|low|As an AuthenticatedUser, I want to rate other bands, so that I can evaluate their quality.|
|US29      |Rate users|low|As an AuthenticatedUser, I want to rate other users, so that I can evaluate their quality.|
|US30      |Chat|medium|As an AuthenticatedUser, I want to chat with other users, so that I can keep contact with them.|
|US31      |Profile Page|high|As an AuthenticatedUser, I want to access other users' profile pages, so that I can see their information and posts.|
|US32      |Band Page|high|As an AuthenticatedUser, I want to access bands' pages, so that I can see their information and posts.|


### Admin
|Identifier|Name|Priority|Description|
| -------- | -- |:------:| --------- |
|US33      |Remove comments|high|As an Admin, I want to remove comments, so that I can remove inappropriate/offensive comments.|
|US34      |Remove posts|high|As an Admin, I want to remove posts, so that I can remove inappropriate/offensive posts.|
|US35      |Warn user|low|As an Admin, I want to warn a user, so that I can warn off users about their negative behaviour so they change it.|
|US36      |Warn band|low|As an Admin, I want to warn a band, so that I can warn off bands with negative behaviour in order to change it.|
|US37      |Ban user|high|As an Admin, I want to remove a user from the system, so that I can remove users with negative behaviour.|
|US38      |Ban band|high|As an Admin, I want to remove a band from the system, so that I can remove bands with negative behaviour.|
|US39      |Give privileges|low|As an Admin, I want to give other users Admin privileges, so that they can help me keep the system safer and cleaner.|
